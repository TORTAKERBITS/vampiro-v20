Trasfondos

El sistema siempre dejará exactamente 3 Trasfondos distintos. Si eliges uno repetido, se completará automáticamente con otro disponible.

2) Méritos y Defectos

Primero debes elegir obligatoriamente 1 Defecto. Después responderás 3 preguntas más para completar 4 rasgos en total. Las preguntas usan lenguaje simple para jugadores nuevos. Los Méritos descuentan puntos gratuitos y los Defectos los aumentan.

3) Hoja editable

Selecciona qué combinación de atributos y habilidades te interesa más para que se asignen automáticamente. Luego puedes modificar Atributos, Habilidades, Virtudes y Trasfondos manualmente, pero cada grupo debe mantener exactamente el total asignado. Las Disciplinas quedan fijas en 1 y la Fuerza de Voluntad es igual a Coraje.

4) Puntos gratuitos

Antes de pasar a historia o impresión, decide si los puntos gratuitos quedarán para conversar con el Narrador o si los asignarás ahora manualmente.

Asignar con el Narrador

Continúa de forma normal. La ficha impresa mantiene los puntos como referencia y no agrega experiencia inicial automática.

Asignar manualmente

Gasta puntos ahora para subir rasgos. Todo punto no gastado se mostrará como Experiencia inicial en la ficha impresa.

Asignación manual: escribe cuántos círculos extra quieres comprar en cada rasgo. El sistema calcula el costo y no permite superar el total disponible. Si intentas gastar de más, ajusta automáticamente el último valor para que la experiencia nunca quede negativa.
